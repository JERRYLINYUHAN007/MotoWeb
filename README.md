# MotoMod 摩托車改裝社群平台

## 專案概述
MotoMod 是一個專業的摩托車改裝社群平台，為車友提供改裝零件展示、技術交流、作品分享和活動參與的綜合服務平台。

## 網站配色方案

網站採用精心設計的配色方案，以優雅、專業的風格展現摩托車改裝社群的特色：

### 主要顏色
- **深咖啡色 (`#534739`)**: 主要強調色，用於按鈕、重要元素和標題
- **淺咖啡色 (`#9E8774`)**: 次要強調色，用於懸停狀態和次要元素
- **淺灰白色 (`#F0F2F1`)**: 淺色背景和文字襯底
- **深藍灰色 (`#1F2D33`)**: 主要文字顏色和深色區域
- **深黑色 (`#0D0D0C`)**: 特殊強調色和陰影

### 功能性顏色
- 文字顏色: 深藍灰色 (`var(--text-color)`)
- 連結顏色: 深咖啡色 (`var(--link-color)`)
- 導航背景: 淺灰白色 (`var(--navbar-bg)`)
- 頁尾背景: 深藍灰色 (`var(--footer-bg)`)
- 主要按鈕: 深咖啡色 (`var(--button-color)`)

### 色彩分離與組織
所有顏色變數都定義在 `colors.css` 文件中，通過 CSS 變數系統實現，確保整個網站保持一致的視覺風格。

## 開發指南

### 安裝步驟
1. 克隆專案：
\`\`\`bash
git clone https://github.com/yourusername/motomod.git

2. 進入專案目錄：

cd motomod

\`\`\`bash
npm install

4. 啟動開發服務器：
\`\`\`bash
npm run dev
\`\`\`

### 圖片爬蟲功能

本專案包含一個爬蟲功能，可以從網路上收集摩托車的原廠和改裝圖片：

1. 爬蟲設置:
   - 安裝爬蟲依賴: `npm install puppeteer axios`
   - 支持車款: SYM SL125, SR125, DRG, MMBCU

2. 執行爬蟲:
\`\`\`bash
npm run scrape
\`\`\`

3. 爬蟲將獲取以下內容:
   - 各車款原廠圖片
   - 各車款改裝整車圖片
   - 各車款不同改裝零件圖片 (排氣管, 前叉, 避震器等)

4. 圖片存儲路徑:
   - `public/uploads/scraped-bikes/[車款]/[類別]/`

5. 注意事項:
   - 請確保遵守版權規定，下載的圖片僅用於個人參考
   - 爬蟲可能需要較長時間處理，請耐心等待
   - 爬蟲可能會被目標網站阻止，請不要頻繁執行
   - 建議設置 .env 文件中的 PUPPETEER_DOWNLOAD_PATH 來自定義 Chrome 下載位置

完整的 API 文檔可通過以下網址訪問：
```

```

主要 API 端點包括：

## 設計理念
- **簡潔優雅**: 運用深咖啡色系創造專業、穩重的視覺形象
- **一致性**: 通過 CSS 變數實現全站配色統一，提高使用者體驗
- **易讀性**: 精心選擇的文字與背景色彩對比度，確保良好的可讀性
- **響應式**: 在不同裝置上保持一致的視覺效果和品牌識別
